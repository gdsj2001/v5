## Purpose

This directory is the independent Z20 v1.5 Vivado hardware project workspace. It was created from the earlier Vivado project as a one-time source copy, but all project files, scripts, reports, and generated outputs for this workspace must stay under `new-vivado/`.

## Hard Boundaries

- Do not edit `vivado_hw_project/`.
- Do not reference files from `vivado_hw_project/` in this project.
- Do not write project-local absolute paths into `.xpr`, `.bd`, `.xci`, Tcl, XDC, README, or generated helper files.
- The active constraint file is `constraints/z20_v1_5_active_mapped.xdc`, with each enabled entry traced back to `../z20-v1_5_20260623.xdc`.
- Do not keep old-project constraint files as active or reference inputs for this workspace. The current constraint sources are `constraints/z20_v1_5_active_mapped.xdc` and `../z20-v1_5_20260623.xdc`.
